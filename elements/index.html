If

Builder 

t.if(m=> m.test == 2, 
    t1 => t1.text("True"),
    t1 => t1.text("False"))

JSX 

<If condition = {m.test == 2} >
    True
    <Else>False</Else>
</If>

Expression 

<tagName>
    {m.test == 2 ? 
        <>True</> 
        : 
        <>False</>
    }
</tagName>

<tagName>{m.test == 2 && <>True not else</>}</tagName>

Foreach

Builder 

t => t.foreach(m => m.items,
    t2 => t2.beginChild("span")
            .text(item => item.myItemProp))
            .endChild();

JSX 

<Foreach src={m.items}>
    <span>{item.myItemProp}</span>
</Foreach>

Expression 

{m.items.forEach(i=> <span>{i.myItemProp}<span/>)}

Warning

Do not use map function of an array, since will behave as expected (transform an array to another array), not bind an array
